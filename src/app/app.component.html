ngx-alert-message

provides an Angular API for simplified use of alerts on your website, completely customizable and easy to implement.

Get Started

Start by installing the Angular Layout library from: npm

Customization

You can customize the position, size and appearance of the alert. Customization can be done for all screen types or different configurations for small screens smaller than or equal to 959px, and for screens larger than or equal to 960px. It is also possible to apply alternative customization for each type of screen.

If you want to use a different customization for small screens, use the property smallScreenConfig: {{ codeInHtmlKeyOpen }} ... {{ codeInHtmlKeyClose }}. Inside it, there will be the same format used for general definitions.

If you want to use a different customization in an alternative theme, set true to property useAlternativeTheme and use the property anternativeTheme: {{ codeInHtmlKeyOpen }} ... {{ codeInHtmlKeyClose }}. Inside it, there will be the same format used for general definitions.

smallScreenConfig

Using ngx-alert-message allows you to use specific customization for small screen devices (small or equal to 959px), such as smartphones. To do this, simply apply the configuration within the smallScreenConfig property

So you can have a default configuration for other screens and one for small screens. See the example of this in the Default Values section.

The smallScreenConfig format is the same as that used in the default configuration, the interface is described in the Customization section.
